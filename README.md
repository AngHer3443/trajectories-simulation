This project models the evolution of dynamic systems using differential equations.
It allows you to analyze how initial conditions such as velocity, position and the way you messure the time affect system's behavior.
it focuses in analysing stability and trajectory intersection between a "persuing" and a "runner" object.

Features:

1-Simulation of trajectories under different initial conditions
2-Adjustable parameters (position, velocity, time)
3-Visualization of system evolution

Aplications:

1-Physics simulation 
2-Motion analysis
